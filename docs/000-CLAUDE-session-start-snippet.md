## Session start rule
Before making any code changes:
1. Read `docs/INFRASTRUCTURE.md`
2. Read `docs/00A-claude-session-start-guide.md`
3. Read `docs/01A-current-build-context.md`
4. Read the relevant section of `docs/09-dev-handoff.md`
5. Summarize what is already live, what is deferred, what step we are on, and which files you plan to edit
6. Wait for confirmation before changing files
